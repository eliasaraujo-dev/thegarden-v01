# The Garden v01

Landing page do The Garden desenvolvida com React + Vite e estilizada com Tailwind CSS.

## Requisitos

- Node.js 20+
- npm 10+

## Executar localmente

1. Instale as dependências:
   - `npm install`
2. Inicie o ambiente de desenvolvimento:
   - `npm run dev`
3. Acesse no navegador o endereço exibido no terminal (geralmente `http://localhost:5173`).

## Build de produção

- Gerar build:
  - `npm run build`
- Visualizar build localmente:
  - `npm run preview`

## Qualidade de código

- Rodar lint:
  - `npm run lint`

## Estrutura principal

- `src/App.jsx`: interface principal e lógica da landing page
- `src/main.jsx`: ponto de entrada da aplicação
- `src/index.css`: estilos globais (Tailwind)
